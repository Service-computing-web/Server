本项目使用golang实现了一个简单的博客网站的服务器。

## Features

- 注册登录
- 查看博客

## Requirements

- go 1.15
- Windows / Linux(Unix like)

## Installation

```shell
$ go get -u github.com/Service-computing-web/Server
```

## Usage

```shell
$ go run main.go
```

请将本项目与对应的客户端配套使用。
